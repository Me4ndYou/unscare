"# unscare" 

-   Clone or download this repo
-   `composer install`
-   `rename .env.example into .env`
-   `php artisan key:generate`

-   Create database
-   `Modify 'DB_DATABASE' on .env file with your DB name.`
-   `php artisan migrate --seed`
-   `npm install`
-   `npm run dev`
-   `php artisan storage:link`
-   `php artisan serve `
